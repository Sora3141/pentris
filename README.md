# PENT! — ペントミノ落ち物パズル

テトリスを **ペントミノ（5 マスのブロック）** に拡張した落ち物パズルゲームです。
4 マスのテトリミノより形のバリエーションが多く、より複雑な戦略が求められます。

## 🔗 リンク

- リポジトリ: https://github.com/Sora3141/pentris
- デモ: https://sora3141.github.io/pentris/

## 概要

- 5 マスからなるブロック（ペントミノ）を落として横一列を消す。
- **HOLD** 機能あり（ブロックの一時保持）。
- 行を消してスコアを稼ぎ、ゲームオーバーまでの高得点を目指す。

## 使用技術

- HTML / CSS / Vanilla JavaScript
- **Canvas 2D** による描画

## 操作（一般的な落ち物パズル準拠）

| キー | 動作 |
|---|---|
| ←→ | 移動 |
| ↑ / Z・X | 回転 |
| ↓ | ソフトドロップ |
| Space | ハードドロップ |
| Shift / C | ホールド |
